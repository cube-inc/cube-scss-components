# Types

## Options

```scss
// To enable responsive font sizes
$enable-responsive-fonts: true !default;
```

## Font sizes

Font sizes are generated from a fixed ratio.

```scss
// Base size
$font-size-base: 1em !default;

// Default font size ratio
$font-size-ratio: $font-size-ratio-minor-third !default;

// Small screen font size ratio
$font-size-ratio-xs: $font-size-ratio-major-second !default;

// Pick one from those popular ratios
$font-size-ratio-minor-second: 1.067; // 15:16
$font-size-ratio-major-second: 1.125; // 8:9
$font-size-ratio-minor-third: 1.2; // 5:6
$font-size-ratio-major-third: 1.25; // 4:5
$font-size-ratio-fourth: 1.333; // 3:4
$font-size-ratio-augmented-fourth: 1.414; // 1:âˆš2
$font-size-ratio-fifth: 1.5; // 2:3
$font-size-ratio-minor-sixth: 1.6; // 5:8
$font-size-ratio-golden: 1.618; // 1:1.618
$font-size-ratio-major-sixth: 1.667; // 3:5
$font-size-ratio-minor-seventh: 1.778; // 9:16
$font-size-ratio-major-seventh: 1.875; // 8:15
$font-size-ratio-octave: 2; // 1:2
```
